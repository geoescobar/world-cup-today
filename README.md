## Installation


1. Clone repo from GitHub

   ```
   gh repo clone geoescobar/world-cup-today
   ```

2. Download dependencies at root level in the integrated terminal

   ```
   npm install
   ```

3. Launch application

   ```
   npm run start
   ```
